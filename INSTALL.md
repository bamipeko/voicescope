# VoiceScope はじめての使い方

音声ファイル（会議録音・取材音声・メモ等）を **AI で自動文字起こし＋要約** するアプリです。
インターネットにデータを送信せずローカルで動かすこともできます。

このページの順に従って、5〜10分でセットアップできます。

---

## 1. ダウンロード

お使いのOSとお好みに合わせて1つ選んでDLしてください:

### Windows をお使いの方

| ファイル | 容量 | 特徴 | おすすめ |
|---|---|---|---|
| `VoiceScope-x.y.z.exe` | ~55 MB | **シングルEXE**（軽量・ブラウザで動く） | ⭐ **初めての方** |
| `VoiceScope-Setup-x.y.z.exe` | ~180 MB | Electron版（トレイ常駐・グローバルショートカット対応） | 常駐させたい方 |

**両方の違い**:
- ⭐ シングルEXE → ダブルクリック → ブラウザのタブで開く（Chromeなど）
- Electron版 → 専用ウィンドウで開く、システムトレイに常駐、`Ctrl+Shift+F8` で録音開始ショートカット使える

迷ったら **シングルEXE** を選んでください。サイズも小さくて軽快です。

### Mac をお使いの方

| ファイル | 容量 |
|---|---|
| `VoiceScope-x.y.z.dmg` | ~100 MB（Apple Silicon / Intel 両対応） |

## 2. インストール

### Windows シングルEXE版

1. ダウンロードした `VoiceScope-x.y.z.exe` をダブルクリック
2. **初回のみ** Windows が「PCを保護しました」と表示することがあります
   - 「詳細情報」をクリック → 「実行」で進めてください
3. 自動的に展開（2〜3秒）→ お使いのブラウザに `localhost:5100` が開く
4. これで完了です

**保存場所** (参考):
- 実行ファイル: 任意の場所
- 展開先: `%LOCALAPPDATA%\VoiceScope-app\` (再展開・更新時に上書き)
- データ: `%APPDATA%\VoiceScope\` (録音・DB・APIキー)

### Windows Electron版

1. `VoiceScope-Setup-x.y.z.exe` をダブルクリック
2. 「PCを保護しました」が出たら「詳細情報」→「実行」
3. インストーラの指示に従ってインストール
4. スタートメニューから `VoiceScope` を起動

### Mac

1. `VoiceScope-x.y.z.dmg` をダブルクリック
2. 開いたウィンドウで `VoiceScope` アイコンを **`Applications` フォルダにドラッグ**
3. `Applications` の `VoiceScope` を **右クリック → 「開く」**
   - ⚠ 通常のダブルクリックだと「開発元を確認できません」と出ます。**初回のみ右クリック→開く** が必要
   - 確認ダイアログで「開く」を押せば起動します
4. 2回目以降は普通にダブルクリックで起動できます

## 3. APIキーを設定する

VoiceScope は **音声→文字起こし** と **文字起こし→要約** の2段階で AI を使います。
それぞれにAPIキーが必要です。

### どこでAPIキーを取れるか

| サービス | 用途 | 取得先 | 料金目安 |
|---|---|---|---|
| **Deepgram** | 文字起こし（推奨） | [console.deepgram.com](https://console.deepgram.com) | 約 ¥40/時間、$200無料クレジット |
| **OpenAI** | 文字起こし＆要約 | [platform.openai.com](https://platform.openai.com/api-keys) | 文字起こし ~¥54/時、要約は安い |
| **xAI (Grok)** | 文字起こし＆要約（新・最安） | [console.x.ai](https://console.x.ai) | 約 ¥15/時 |
| **Gemini** | 要約のみ | [aistudio.google.com](https://aistudio.google.com/app/apikey) | 無料枠多い |
| **Anthropic** | 要約のみ | [console.anthropic.com](https://console.anthropic.com) | gpt-5-mini と同程度 |

**最低限**: 文字起こし用に Deepgram または OpenAI のどちらか1つ + 要約用に OpenAI / Gemini / Grok のどれか1つ。
**コスパ重視なら**: xAI Grok だけ登録すれば文字起こし＋要約両方カバーできて最安。

### キーの貼り付け方

1. VoiceScope を起動した画面の左メニューから **「設定」** をクリック
2. 「APIキー（クラウドモード）」の欄に各サービスのキーを貼り付け
3. 右の **「保存」** ボタンをクリック
4. キーは暗号化されて自分のPCに保存されます（外部送信なし）

> **トライアルコードをお持ちの場合**: APIキーは不要です。
> 設定の「プラン」セクションで配布されたコードを入力してください。
> 14日間、運営側のAPIキーで全機能が使えます。

## 4. 試しに録音してみる

### マイク録音

1. 画面右下の 🔴 **赤いマイクボタン** をクリック → 録音開始
2. 話す
3. もう一度 🔴 ボタン → 録音停止
4. 自動で「文字起こし → 整形 → 要約」が走ります（30秒〜数分）

### 既存の音声ファイルから

1. ダッシュボード右上 **「ファイルをアップロード」** をクリック
2. mp3 / wav / m4a / webm / ogg / flac ファイルを選択
3. 自動処理開始

### テキストファイルから（既に文字起こし済みの場合）

1. **「ファイルをアップロード」** で `.txt` / `.md` ファイルを指定
2. 自動でテキスト解析→要約

## 5. 主な機能

| 機能 | 説明 |
|---|---|
| 🎤 録音 | マイク + システム音声を同時録音可（会議の自動録音にも対応） |
| 📝 文字起こし | Deepgram / OpenAI Whisper / Grok / ローカル(whisper.cpp) から選択 |
| ✨ 整形 | フィラー除去・誤字修正（gpt-5-nano推奨、自動フォールバック付き） |
| 📊 要約 | テンプレート選択 or カスタムプロンプトで自由要約 |
| 🔍 横断質問 (Ask All) | 複数録音にまたがる質問・分析 |
| 📦 アーカイブ / 🗑 ゴミ箱 | 削除前にアーカイブ、ゴミ箱は X 日後自動削除（設定で1〜30日） |
| 🔒 ローカル保護 | 横断検索の対象外にできる（プライバシー） |
| 📁 エクスプローラで開く | 録音詳細画面から音声ファイルの場所をすぐ開ける |

## 6. データはどこに保存される？

すべて**自分のPCにのみ**保存されます（クラウドサービスはAIで処理する瞬間だけ通信）。

```
Windows: %APPDATA%\VoiceScope\
Mac:     ~/Library/Application Support/VoiceScope/

  ├── voicescope.db        ← 文字起こし・要約・タグ・設定（SQLite）
  └── audio/               ← 録音音声（録音ごとに1ファイル）
       ├── abc123.webm
       └── ...
```

録音詳細ページの **「📁 エクスプローラで開く」** ボタンで、いつでもファイルの場所を確認できます。

## 7. 困ったとき

### 起動しない / 真っ白な画面

- **再起動してみる**: タスクトレイに残っていることがあるので、タスクマネージャーで `voicescope-server.exe` を終了してから再度起動
- **別ポートで動いてしまっている**: `localhost:5100` を別アプリが使っている可能性。他のローカルサーバーを停止してください
- **Mac で開かない**: `右クリック → 開く` を試してください

### APIエラーが出る

設定画面でAPIキーが正しく設定されているか、APIキーの残高があるかを確認してください。
要約失敗時は自動でローカルLLM（Ollama）にフォールバックする設定もあります（設定→「整形」セクション）。

### 録音されない / マイクが反応しない

- 初回起動時にブラウザがマイク許可を求めます。「許可」を選んでください
- 拒否してしまった場合: ブラウザのアドレスバー左の🔒アイコン → サイト設定 → マイクを「許可」

### もっと詳しい操作を知りたい

各画面の `?` アイコン や設定画面の説明文を見てください。それでも解決しなければ配布元へお問い合わせを。

## 8. アンインストール

### Windows シングルEXE版

```powershell
# プログラム本体
del VoiceScope-x.y.z.exe

# 展開先（再展開キャッシュ）
rmdir /s /q "%LOCALAPPDATA%\VoiceScope-app"

# データ（録音・文字起こし・APIキー）も消したい場合
rmdir /s /q "%APPDATA%\VoiceScope"
```

### Windows Electron版

通常のアンインストール（コントロールパネルから）+ `%APPDATA%\VoiceScope` 削除。

### Mac

```bash
rm -rf "/Applications/VoiceScope.app"
rm -rf "$HOME/Library/Application Support/VoiceScope"
rm -rf "$HOME/Library/Logs/VoiceScope"
```
